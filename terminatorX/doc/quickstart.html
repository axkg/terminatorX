1.2. Quickstart

  1. If you don't have LADSPA (Linux Audio Developer's Simple Plugin API) installed, go and get it from the LADSPA-homepage and install the SDK. terminatorX requires ladspa.h.

  2. Install terminatorX (see INSTALL)

  3. Run terminatorX

  4. Load an audiofile into the available turntable (you can add more turntables if you want) (Hint: Click on the button that holds "NONE" to load an audiofile, or drop one over the audiofile display from your filemanager if your filemanager is efm, gmc, Nautilus or compatible (kfm does not work))

  5. Load a LADSPA plugin of your choice by clicking the turntable's FX button and enable it.

  6. Click on the "Audio Engine"-button to start playback.

  7. Click on the "Mouse-Grab"-button to enter grab mode. The turntable with the red border has the input focus.

  8. Press SPACE or left mouse button to scratch.

  9. Release SPACE or left mouse button to let the scratch-file run at the turntable's default speed.

  10. Press ESCAPE to quit grab mode.
